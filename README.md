# Student Report Card Management System (C)

A menu-driven console application developed in C language using file handling
to manage student academic records.

## Features
- Add student record
- Display report cards
- Automatic total, percentage & grade calculation
- File-based permanent storage

## Technologies Used
- C Language
- File Handling (Binary Files)

## Project Structure
- src/  -> Source code
- data/ -> Auto-generated data files

## How to Run
gcc src/student_report_card.c -o report  
./report
